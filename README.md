# アンカーパチンコ・シミュレータ (Anchor Pachinko Simulator)

Cプログラミング雑誌のパズル問題「アンカーパチンコ」の挙動を視覚的に再現し、解答を検証するためのWebシミュレータアプリケーションです。

---

## ⚓ 概要

本プロジェクトは、上部から落としたボールが「いかり（アンカー）」のピンに当たるたびに向きをカタカタと左右に変えながら落ちるパチンコ台の挙動をシミュレートします。
ブラウザ上でボールの滑らかな落下アニメーションといかりの回転トランジションを視覚的に確認しながら、パズルの解答が正しいかを手動および自動キューで検証できます。

### 🌟 主な機能
* **リアルタイム物理シミュレーション**: ボールがいかりの根元に触れたその瞬間に、いかりのトグル回転アニメーションと軌道変化が完全同期します。
* **落下速度調整スライダー**: アニメーション中であっても、落下速度を極低速（スローモーションでの検証用）から高速までリアルタイムに加減速できます。
* **2種類の台（Fig 1 / Fig 3）を搭載**: タブ切り替えによって、8個のいかりを持つ「Fig. 1の台」と、18個のいかりを持つ「Fig. 3の台」を瞬時に切り替えられます。
* **解答検証ショートカット (自動投入キュー)**: 最小手数の解答を自動でセットし、順次1個ずつ落としてすべてのいかりが「左（L）」に倒れる様子をオートでデモ検証できます。
* **リアルタイムログ**: ボールがいかりに当たるたびに、接触ログがコンソールにリアルタイムで流れます。

---

## 🎮 使い方

本アプリケーションはサーバーなどの環境構築を必要とせず、スタンドアロンで動作します。

1. 本リポジトリをクローンまたはダウンロードします。
2. フォルダ内の **`index.html`** をブラウザ（Google Chrome, Microsoft Edge, Safari など）でダブルクリックして開きます。
3. 画面上のコントロールパネルより、投入口ボタンをクリックしてボールを落とすか、「解答検証ショートカット」から自動デモを実行してください。
